IDE 列表

最后更新：2016 年 9 月 16 日

跳转至：C / C++    C#    Java    Lisp    Python    PHP    Perl    Clojure    Rust

这个页面是专门为完全的编程初学者所准备的，以下列举一些各编程语言的著名集成开发环境 (IDE)。

有标准又好用的 IDE，可是下面没列出吗？欢迎加入“一起学编程！Go！”面簿群组，并在那里提出推荐。

C / C++

  • Code::Blocks (Windows/Linux/macOS)
    • http://www.codeblocks.org/downloads/26
     
  • Dev-C++ (Windows)
    • http://orwelldevcpp.blogspot.com
     
  • Visual Studio (Windows)
    • 2013 和 2015 版本，请看这里，你也可以通过 DreamSpark 或者 MSDN 获取。

    如何设置可以使用 C 和 C++ 标准函数？请看这里。
     
  • CppDroid (Android)
    • http://www.cppdroid.info
     
  • CppCode (iOS)
    • http://www.cppcode.info
     
  • C-Free (Windows)
    • http://www.programarts.com

C#

  • Visual Studio (Windows)
    • 2013 和 2015 版本，请看这里，你也可以通过 DreamSpark 或者 MSDN 获取。
     
  • Xamarin for Visual Studio (Windows)/Xamarin Studio (macOS)
    • http://xamarin.com/download
     
  • SharpDevelop (Windows)
    • http://www.icsharpcode.net/OpenSource/SD/Download

Java

  • 重要！Java SE Development Kit 8 (Windows/Linux/macOS)
    • http://www.oracle.com/technetwork/java/javase/downloads/jdk8-downloads-2133151.html

    这个是安装任何 IDE 之前，必须安装的编译器组件。

    API 文件下载：http://www.oracle.com/technetwork/java/javase/documentation/jdk8-doc-downloads-2133158.html
     
  • 重要！OpenJDK (Windows/Linux)
    • http://openjdk.java.net/install

    OpenJDK 可以用来替代以上的商业版 Java DK，任选其一。

    Red Hat 制作了适用于 Windows 的安装包，可在这里下载：http://developers.redhat.com/products/openjdk/overview
     
  • JCreator (Windows)
    • http://www.jcreator.org/download.htm

    Pro 4.50 破解版，请看本群组文档。
     
  • NetBeans (Windows/Linux/macOS)
    • https://netbeans.org/downloads

    这个 IDE 也支持 C 和 C++ 的编译，还有 PHP 的解析。
     
  • Eclipse (Windows/Linux/macOS)
    • http://www.eclipse.org/downloads

    这个 IDE 也支持 C 和 C++ 的编译，还有 PHP 的解析，不过需另外下载。
     
  • IntelliJ IDEA (Windows/Linux/macOS)
    • https://www.jetbrains.com/idea/download

    永久激活方法：安装之后提示继续，选择 License Server，输入 http://idea.lanyus.com/ 作为网址，最后按 OK。
     
  • jGRASP (Windows/Linux/UNIX/macOS)
    • http://spider.eng.auburn.edu/user-cgi/grasp/grasp.pl?;dl=download_jgrasp.html
     
  • AIDE (Android)
    • http://www.android-ide.com

    这个 IDE 也支持 C 和 C++ 的编译，还有网页相关 (HTML、CSS 和 JavaScript) 开发。

Lisp

  • Corman Lisp (Windows)
    • http://www.cormanlisp.com

    第一个 Windows 下能编译成 C 语言大小的 Lisp 编译器，今年年头开源。
     
  • SBCL (Windows/Linux/macOS)
    • http://www.sbcl.org

    高效率的 native Lisp 编译器，能产生高效的 Lisp 二进制包。
     
  • ECL (Windows/Linux/macOS)
    • https://gitlab.com/groups/embeddable-common-lisp

    能编译成嵌入式系统的二进制文件，是 Lisp-to-C compiler 的变种。

Python

  • 重要！Python (Windows/Linux/UNIX/macOS)
    • https://www.python.org/downloads

    这个是安装任何 IDE 之前，必须安装的编译器组件；其中有自带命令处理程序和简单的 IDE。
     
  • PyCharm (Windows/Linux/macOS)
    • https://www.jetbrains.com/pycharm
     
  • PyPy (Windows/Linux/macOS)
    • http://pypy.org

    Python 2.7/3.2.5 JIT 解释器，用 Python 写的 Python 解释器，更加快速 (2.5 倍的效率增长)。

PHP

  • 重要！PHP (Linux/UNIX/macOS)
    • http://php.net/downloads.php

    这个是安装任何 IDE 之前，必须安装的解析器组件。
     
  • 重要！PHP for Windows (Windows)
    • http://windows.php.net/download

    这个是安装任何 IDE 之前，必须安装的解析器组件。
     
  • PhpStorm (Windows/Linux/macOS)
    • https://www.jetbrains.com/phpstorm

    永久激活方法：安装之后提示继续，选择 License Server，输入 http://idea.lanyus.com/ 作为网址，最后按 OK。

Perl

  • ActivePerl (Windows/Linux/macOS)
    • http://www.activestate.com/activeperl/downloads

    这个是解析器组件。

Clojure

  • Light Table (Windows/Linux/macOS)
    • http://lighttable.com

    高定制，能内运算，最佳的 Clojure IDE。
     
  • Leiningen (Windows/Linux)
    • https://leiningen.org

Rust

目前还没有人为 Rust 开发单独的 IDE，所以只提供 IDE 的插件。

  • RustDT
    • https://github.com/RustDT/RustDT/blob/latest/documentation/Installation.md

    适用于 Eclipse。
     
  • Visual Rust
    • https://github.com/PistonDevelopers/VisualRust/blob/master/README.md

    适用于 Visual Studio 2013 或以上。
