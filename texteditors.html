文字编辑器列表

最后更新：2016 年 9 月 9 日

有方便使用的文字编辑器，可是下面没列出吗？欢迎加入“一起学编程！Go！”面簿群组，并在那里提出推荐。

  • Sublime Text 3 (Windows/Linux/macOS)
    • http://www.sublimetext.com/3
     
  • Visual Studio Code (Windows/Linux/macOS)
    • https://code.visualstudio.com/download
     
  • Atom (Windows/Linux/macOS)
    • https://github.com/atom/atom/releases/latest
     
  • GNU Emacs (Windows/Linux/macOS)
    • https://www.gnu.org/software/emacs/download.html
     
  • GNU nano (Linux)
    • https://www.nano-editor.org/download.php

    绝大部分的 Linux 发行版都自带这个编辑器。
     
  • Vim (Windows/Linux/UNIX/macOS/Android/iOS)
    • http://www.vim.org/download.php

    绝大部分的 Linux 发行版都自带这个编辑器。
     
  • Notepad++ (Windows)
    • https://notepad-plus-plus.org/zh/download
     
  • Kilo (Linux/macOS)
    • https://github.com/antirez/kilo

    据称是世上最小的文字编辑器。下载后，请自行编译。
